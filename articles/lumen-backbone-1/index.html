Lumen

Lumen 是從 Laravel 切割出來的 micro-framework，並非用來取代 Laravel，基本上 Lumen 有很多元件仍來自於 Lavavel，主要適用於 Micro Service 和 RESTFul API ，最大賣點是速度。
